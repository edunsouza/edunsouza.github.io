Política de privacidade para TCC - Graph API

Todas as suas informações pessoais recolhidas, serão usadas para o ajudar a tornar a sua visita no nosso perfil do Facebook mais produtiva e agradável.

A garantia da confidencialidade dos dados pessoais dos utilizadores do nosso site é importante para o TCC - Graph API.

Todas as informações pessoais relativas a membros, assinantes, clientes ou visitantes que usem o TCC - Graph API serão tratadas em concordância com a Lei da Proteção de Dados Pessoais de 26 de outubro de 1998 (Lei n.º 67/98).

A informação pessoal recolhida pode incluir o seu nome e sua descrição de biografia no Facebook.

O uso do TCC - Graph API pressupõe a aceitação deste Acordo de privacidade. O autor do TCC - Graph API reserva-se ao direito de alterar este acordo sem aviso prévio. Deste modo, recomendamos que consulte a nossa política de privacidade com regularidade de forma a estar sempre atualizado.

As publicações

As publicações recolhidas são apenas utilizadas para a apresentação de Trabalho de Conclusão de Curso na feira da semana acadêmica da Faculdade Fatec Senai Porto Alegre.

Expiração

Este projeto é voltado apenas para construção e apresentação do Trabalho de Conclusão de Curso para obter o título de Tecnólogo em Análise e Desenvolvimento de Sistemas

Ligações a Sites de terceiros

O TCC - Graph API não possui ligações com outros sites, apenas o Facebook.
